gaithub

GitHub for GAIT repos (Git for AI Turns).

What are GAIT + GaitHub?

GAIT is Git-like version control for AI conversations: turns become objects, commits build a DAG, branches let you explore “what-if” timelines, and pinned memory keeps critical context from drifting.

GaitHub is the remote store for GAIT: users create accounts, create repos, push/pull objects + refs, fork repos, collaborate, and open/merge pull requests.
